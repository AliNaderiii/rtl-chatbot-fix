# RTL Chatbot Fix - راست‌چین‌ساز چت‌بات‌ها

<div dir="rtl" align="right">

## 🎯 مشکل چیست؟

وقتی از چت‌بات‌هایی مثل ChatGPT، Claude، Gemini و غیره استفاده می‌کنید و پاسخ به زبان فارسی، عربی، کردی، اردو، عبری یا هر زبان راست‌به‌چپ دیگر داده می‌شود، متن به هم می‌ریزد:

- متن به سمت چپ می‌چسبد به جای راست
- علائم نگارشی در جای اشتباه قرار می‌گیرند
- ترکیب متن فارسی و انگلیسی به هم می‌ریزد
- خواندن متن سخت و آزاردهنده می‌شود

## ✨ راه‌حل

این اکستنشن مرورگر به طور خودکار متون راست‌به‌چپ را در تمام چت‌بات‌های معروف تشخیص داده و به درستی راست‌چین می‌کند.

## 🚀 نصب و راه‌اندازی

### روش اول: نصب از فروشگاه کروم (به زودی)
منتظر انتشار در Chrome Web Store باشید.

### روش دوم: نصب دستی (Developer Mode)

1. این پوشه را دانلود و از حالت فشرده خارج کنید
2. مرورگر Chrome یا Edge را باز کنید
3. به آدرس `chrome://extensions` بروید
4. گزینه **Developer mode** (حالت توسعه‌دهنده) را در بالای صفحه فعال کنید
5. روی **Load unpacked** (بارگذاری بسته‌نشده) کلیک کنید
6. پوشه `rtl-chatbot-fix` را انتخاب کنید
7. تمام! اکستنشن فعال است.

## ⚙️ تنظیمات

از طریق آیکون اکستنشن در نوار ابزار مرورگر می‌توانید:

- **فعال/غیرفعال کردن:** اکستنشن را روشن یا خاموش کنید
- **آستانه تشخیص:** میزان حساسیت تشخیص متن راست‌به‌چپ را تنظیم کنید (۱۰٪ تا ۸۰٪)
- **اسکن مجدد:** صفحه فعلی را دوباره اسکن کنید

## 🛠️ نحوه کار

1. اکستنشن متن پیام‌های چت‌بات را بررسی می‌کند
2. با استفاده از محدوده‌های Unicode، حروف راست‌به‌چپ را تشخیص می‌دهد
3. اگر درصد حروف راست‌به‌چپ از آستانه تنظیم شده بیشتر باشد، متن راست‌چین می‌شود
4. بلوک‌های کد، ریاضیات و LaTeX به صورت چپ‌به‌راست باقی می‌مانند
5. متن‌های در حال تایپ (streaming) نیز به صورت زنده پردازش می‌شوند
